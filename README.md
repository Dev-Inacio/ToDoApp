# ToDoApp

Este é um aplicativo simples de lista de tarefas (ToDoApp), desenvolvido utilizando HTML, CSS (Tailwind) e JavaScript.

## Funcionalidades

- Adicionar tarefas
- Editar tarefas
- Excluir tarefas
- Marcar tarefas como concluídas
- Armazenar dados no localStorage
